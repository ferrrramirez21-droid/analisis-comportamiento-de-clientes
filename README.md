# analisis-comportamiento-de-clientes

Este proyecto analiza el comportamiento de compra de los clientes utilizando datos transaccionales de 3,900 compras en diversas categorías de productos. El objetivo es descubrir información sobre patrones de gasto, segmentos de clientes, preferencias de productos y comportamiento de suscripción para apoyar la toma de decisiones estratégicas de negocio.
